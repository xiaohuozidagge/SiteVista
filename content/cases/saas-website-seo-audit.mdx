This example audit examines a B2B SaaS website. The site offers project management software and targets mid-market technology companies.

## What We Found

The technical foundation was generally solid, but several issues were limiting organic visibility.

### Technical Issues

The site had a clean architecture, but page speed on mobile was significantly below benchmarks. Several JavaScript resources were render-blocking, and image optimization was inconsistent.

### Content Gaps

The blog had good depth in product-related topics but missed opportunities in top-of-funnel educational content that potential customers search for before they are ready to evaluate tools.

### Keyword Opportunities

Competitor analysis revealed several high-volume, low-competition keywords that the site was not targeting at all.

## Recommendations

1. **Optimize Core Web Vitals** — Address LCP on mobile by optimizing images and deferring non-critical JavaScript
2. **Build top-of-funnel content** — Create guides targeting informational queries in the project management space
3. **Implement structured data** — Add Organization and Software Application schema
4. **Fix meta descriptions** — Write unique, compelling descriptions for all key landing pages
